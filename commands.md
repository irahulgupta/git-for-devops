# Git Commands Documentation

## General Commands
- `ls`               # List directory contents
- `clear`            # Clear the terminal screen
- `pwd`              # Print working directory

## Directory Management
- `mkdir git-for-devops`  # Create a new directory
- `cd git-for-devops`     # Change directory

## File Operations
- `vim hello-dosto.txt`   # Open file in Vim editor
- `touch nibba.txt`        # Create a new file
- `touch nibbi.txt`        # Create another new file
- `rm nibbi.txt`          # Remove a file
- `git restore nibbi.txt`  # Restore a deleted file
- `vim nibbi.txt`          # Open the file in Vim editor
- `touch nibbu.txt`        # Create another new file
- `vim nibbu.txt`          # Open file in Vim editor

## Git Initialization
- `git init`               # Initialize a new Git repository

## Git Status and Tracking
- `git status`             # Show the working tree status
- `git add <file>`         # Stage changes for the next commit
- `git commit -m "<message>"`  # Commit changes with a message
- `git log`                # Show commit logs
- `git log --oneline`      # Show commit logs in a concise format

## Branching and Merging
- `git checkout -b dev`    # Create and switch to a new branch
- `git checkout master`     # Switch to the master branch
- `git checkout dev`       # Switch to the dev branch
- `git branch`              # List all branches

## Configuration
- `git config --global user.name "<name>"`    # Set global username
- `git config --global user.email "<email>"`  # Set global email

## Miscellaneous
- `ls -a`                  # List all files, including hidden ones
- `clear`                  # Clear the terminal screen
